# Setup de Analytics - Frontend

## Instalação de Dependências

```bash
npm install recharts
```

## Componentes Criados

1. **MotoresChart** - Gráfico de pizza da distribuição de motores
2. **AreasSilenciadasHeatmap** - Heatmap das áreas mais silenciadas
3. **TimelineChart** - Gráfico de linha temporal de diagnósticos
4. **DashboardKPIs** - Cards com KPIs principais

## Uso

Importe os componentes no dashboard:

```tsx
import { MotoresChart, AreasSilenciadasHeatmap } from '@/components/dashboard';
```
